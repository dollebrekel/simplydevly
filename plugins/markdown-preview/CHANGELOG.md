# Changelog

## [1.0.0] — 2026-04-21

### Added

- Initial release of markdown-preview plugin
- Markdown rendering via siplyui.MarkdownView (headings, code, lists, tables, etc.)
- Subscribes to FileSelectedEvent for automatic file preview
- Placeholder shown when no .md file is selected
- Lazy-load: no filesystem I/O until first panel activation
